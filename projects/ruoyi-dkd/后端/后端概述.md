# 后端概述

## 技术栈

- **框架**：Spring Boot
- **ORM**：MyBatis
- **数据库**：MySQL
- **构建工具**：Maven

## 模块结构

```
dkd-parent/
├── dkd-admin/        # 主启动模块（Controller 层）
├── dkd-common/       # 公共模块（工具类、常量、异常）
├── dkd-framework/    # 框架核心（安全、配置、拦截器）
└── dkd-system/       # 系统模块（Service、Mapper、实体类）
```
