# 前端概述

## 技术栈

- **框架**：Vue 3
- **构建工具**：Vite
- **UI 组件库**：Element Plus
- **包管理器**：pnpm
- **路由**：Vue Router
- **状态管理**：Pinia
- **HTTP 请求**：Axios

## 目录结构

```
dkd-vue/
├── src/
│   ├── api/          # 接口请求模块
│   ├── assets/       # 静态资源（图片、样式）
│   ├── components/   # 公共组件
│   ├── directive/    # 自定义指令
│   ├── layout/       # 页面布局（侧边栏、顶栏等）
│   ├── plugins/      # 插件注册
│   ├── router/       # 路由配置
│   ├── store/        # Pinia 状态管理
│   ├── utils/        # 工具函数（请求封装、权限等）
│   └── views/        # 页面视图
├── vite.config.js    # Vite 配置
├── index.html        # 入口 HTML
└── package.json
```

## 关键文件

| 文件 | 作用 |
|---|---|
| `src/main.js` | 应用入口，注册插件和全局组件 |
| `src/permission.js` | 路由守卫，控制页面访问权限 |
| `src/settings.js` | 全局配置（标题、主题等） |
| `src/App.vue` | 根组件 |
| `vite.config.js` | Vite 构建配置（代理、别名等） |
