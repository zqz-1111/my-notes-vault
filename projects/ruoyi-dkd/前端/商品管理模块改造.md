### 2.2 商品管理改造

#### 价格单位转换（分 ↔ 元）

数据库存储价格单位是**分**，前端展示和输入用**元**：

```html
<!-- 表格展示：price / 100 显示元 -->
<el-table-column label="商品价格" align="center" prop="price">
  <template #default="scope">
    <el-tag>{{ scope.row.price / 100 }}元</el-tag>
  </template>
</el-table-column>

<!-- 表单输入：el-input-number 输入元 -->
<el-form-item label="商品价格" prop="price">
  <el-input-number :min="0.01" :max="999.99" :precision="2" :step="0.5" v-model="form.price" /> 元
</el-form-item>
```

脚本中做单位转换：

```javascript
// 修改时：分 → 元（回显）
function handleUpdate(row) {
  reset();
  const _skuId = row.skuId || ids.value;
  getSku(_skuId).then(response => {
    form.value = response.data;
    form.value.price = form.value.price / 100; // 分 → 元
    open.value = true;
  });
}

// 提交时：元 → 分（保存）
function submitForm() {
  proxy.$refs["skuRef"].validate(valid => {
    if (valid) {
      form.value.price = Math.round(form.value.price * 100); // 元 → 分
      // ... 调用接口
    }
  });
}
```

**关键点**：
- `Math.round()` 避免 `0.1 * 100 = 10.000000000000002` 这类浮点精度问题
- el-input-number 的 `:precision="2"` 限制输入两位小数，`:step="0.5"` 步进 0.5 元

#### 商品类型下拉选择

表格中用 v-for 匹配显示商品类型名称：

```html
<el-table-column label="商品类型" align="center" prop="classId">
  <template #default="scope">
    <div v-for="item in skuClassList" :key="item.classId">
      <span v-if="item.classId == scope.row.classId">{{ item.className }}</span>
    </div>
  </template>
</el-table-column>
```

表单中用 el-select 下拉选择：

```html
<el-form-item label="商品类型" prop="classId">
  <el-select v-model="form.classId" placeholder="请选择商品类型" style="width: 100%">
    <el-option v-for="item in skuClassList" :key="item.classId" :label="item.className" :value="item.classId" />
  </el-select>
</el-form-item>
```

注意 classId 的验证规则 trigger 要改为 `"change"`（下拉选择触发）而不是 `"blur"`（输入框触发）。

#### 创建时间格式补时分秒

```html
<span>{{ parseTime(scope.row.createTime, '{y}-{m}-{d} {h}:{i}:{s}') }}</span>
```

## 三、踩坑避坑指南

**坑 1：浮点精度问题**

```javascript
0.1 + 0.2 = 0.30000000000000004  // 不是 0.3
0.1 * 100 = 10.000000000000002   // 不是 10
```

解决方法：用 `Math.round()` 四舍五入取整。

**坑 2：el-input-number 的 step 属性**

- `:step="0.5"` 表示点击上下箭头时每次增减 0.5
- 但手动输入不受 step 限制，只受 min/max 约束
