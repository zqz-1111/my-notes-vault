## 通用模式速查

| 模式 | 用法 | 场景 |
|---|---|---|
| 序号列 | `type="index"` | 所有表格 |
| 下拉选择 | `el-select` + `v-for` | 外键字段（区域、角色、类型等） |
| 嵌套对象 | `prop="region.regionName"` | 后端返回嵌套对象 |
| v-for 匹配 | `v-for` + `v-if` | 后端只返回 ID，前端匹配名称 |
| 条件渲染 | `v-if="form.id!=null"` | 新增 vs 编辑显示不同字段 |
| 价格转换 | `price / 100` 展示，`Math.round(price * 100)` 保存 | 数据库存分，页面显示元 |
| 字典组件 | `<dict-tag :options="xxx" :value="xxx"/>` | 状态、类型等字典字段 |
| 查看详情 | 复用 `loadAllParams` + 动态添加筛选条件 | 关联数据列表 |
