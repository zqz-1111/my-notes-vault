# 策略管理模块改造

## 一、写在前面

1. 为什么学这个：策略管理页面需要 el-input-number 整数输入和查看详情功能
2. 学完你能掌握：el-input-number 精度控制、查看详情复用 loadAllParams

## 二、正文

### 2.1 表格改造

- 序号列：`type="index"` 自动生成递增序号
- 策略方案：直接显示 discount 字段
- 创建时间：`parseTime` 格式化带时分秒
- 操作按钮：查看详情 + 修改 + 删除，去掉 icon

### 2.2 el-input-number 整数输入

折扣比例用数字输入框，限制 1-100 的整数：

```html
<el-form-item label="策略方案" prop="discount">
  <el-input-number :min="1" :max="100" :precision="0" v-model="form.discount" placeholder="请输入策略" />
</el-form-item>
```

**关键点**：`:precision="0"` 表示不允许小数，只输入整数。

### 2.3 策略查看详情

点击"查看详情"显示该策略下的设备列表：

```javascript
import { listVm } from "@/api/manage/vm";
import { loadAllParams } from "@/api/page";

const policyOpen = ref(false);
const vmList = ref([]);

function getPolicyInfo(row) {
  form.value = row;
  loadAllParams.policyId = row.policyId;
  listVm(loadAllParams).then(response => {
    vmList.value = response.rows;
    policyOpen.value = true;
  });
}
```

对话框：

```html
<el-dialog v-model="policyOpen" title="策略详情" width="500px">
  <el-form-item label="策略名称" prop="policyName">
    <el-input v-model="form.policyName" disabled />
  </el-form-item>
  <label>包含设备：</label>
  <el-table :data="vmList">
    <el-table-column label="序号" type="index" width="80" align="center" />
    <el-table-column label="点位地址" align="left" prop="addr" show-overflow-tooltip />
    <el-table-column label="设备编号" align="center" prop="innerCode" />
  </el-table>
</el-dialog>
```
