# ruoyi-dkd 项目概述

## 项目简介

基于若依（RuoYi）框架的点餐系统（dkd），前后端分离架构。

## 技术栈

| 层 | 技术 |
|---|---|
| 前端 | Vue 3 + Vite + Element Plus |
| 后端 | Spring Boot + MyBatis |
| 数据库 | MySQL |
| 项目管理 | Maven（后端）/ pnpm（前端） |

## 项目结构

```
ruoyi-dkd/
├── dkd-vue/          # 前端（Vue 3 + Vite）
│   ├── src/
│   │   ├── api/          # 接口请求
│   │   ├── assets/       # 静态资源
│   │   ├── components/   # 公共组件
│   │   ├── layout/       # 布局组件
│   │   ├── router/       # 路由
│   │   ├── store/        # 状态管理（Pinia）
│   │   ├── utils/        # 工具函数
│   │   └── views/        # 页面视图
│   └── vite.config.js
└── dkd-parent/       # 后端（Spring Boot）
    ├── dkd-admin/        # 主启动模块
    ├── dkd-common/       # 公共模块
    ├── dkd-framework/    # 框架核心
    └── dkd-system/       # 系统模块
```
