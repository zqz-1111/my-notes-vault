# 技术方案

## 一、整体架构

```
数据来源：TCMSP + 中医文献
        ↓
数据预处理：清洗、结构化、标注
        ↓
命名实体识别（NER）：识别中药/方剂/疾病/症状/成分
        ↓
关系抽取（RE）← 改进点：融合药理学特征的 BERT 关系抽取
        ↓
知识图谱构建：三元组 → Neo4j 存储
        ↓
可视化展示 + 查询验证
```

## 二、数据来源

### 2.1 TCMSP 数据库（主数据源）

- 网址：https://tcmsp-e.com
- 规模：499 味中药、29,384 种成分、3,311 个靶点、837 种疾病
- 核心字段：
  - 中药名 → 成分（化合物）
  - 成分 → 靶点（蛋白质）
  - 靶点 → 疾病
  - 每个成分有 OB（口服生物利用度）、DL（类药性）等药理学参数

### 2.2 TCMID 2.0（补充数据源）

- 规模：46,929 首方剂、8,159 味草药、43,413 种成分、4,633 种疾病
- 用途：补充方剂-中药组成关系

### 2.3 中医文献/教材

- 来源：《方剂学》《中药学》教材文本
- 用途：NER/RE 模型训练的标注语料

## 三、实体类型设计

| 实体类型 | 说明 | 示例 |
|---|---|---|
| Herb（中药） | 单味中药 | 黄芪、当归、人参 |
| Formula（方剂） | 中药处方 | 四物汤、六味地黄丸 |
| Component（成分） | 中药化学成分 | 槲皮素、黄芩苷 |
| Disease（疾病） | 疾病名称 | 冠心病、糖尿病 |
| Symptom（症状） | 临床症状 | 气短、乏力、失眠 |
| Target（靶点） | 蛋白靶点 | PTGS2、ESR1 |

## 四、关系类型设计

| 关系类型 | 头实体 | 尾实体 | 示例 |
|---|---|---|---|
| contains | Formula | Herb | 四物汤 → contains → 当归 |
| has_component | Herb | Component | 黄芪 → has_component → 槲皮素 |
| treats | Herb/Formula | Disease | 黄芪 → treats → 气虚 |
| targets | Component | Target | 槲皮素 → targets → PTGS2 |
| indicates | Disease | Symptom | 气虚 → indicates → 乏力 |
| has_property | Herb | 属性 | 黄芪 → 性温 → 味甘 |

## 五、创新点设计（核心）

### 改进方案：融合药理学特征的中医文本关系抽取

**现有方法的问题：**
- 通用 BERT 模型对中医领域文本理解不足
- 仅依赖文本语义，忽略了中药的药理学特征（OB、DL、分子量等）
- 中医文本中实体边界模糊，关系类型多样

**改进思路：**
1. 在 BERT 编码器基础上，加入药理学特征嵌入层
2. 将 TCMSP 中的 OB、DL 等数值特征作为辅助信息融入关系分类
3. 用注意力机制动态融合文本特征和药理学特征

**模型结构：**
```
输入：中医文本 + 实体对
        ↓
BERT 编码器 → 文本语义向量
        ↓                    ↓
药理学特征嵌入层 ← OB/DL/分子量等数值特征
        ↓
特征融合层（注意力机制）
        ↓
关系分类器 → 关系类型
```

**创新点表述（论文用）：**
"提出一种融合药理学特征的中医文本关系抽取方法，在 BERT 预训练模型基础上，通过注意力机制动态融合文本语义特征与中药药理学特征（口服生物利用度、类药性等），有效提升了中医领域关系抽取的准确率。"

## 六、实验方案

### 6.1 数据集

| 数据集 | 用途 | 规模 |
|---|---|---|
| TCMSP 结构化数据 | 知识图谱构建 | 499 中药、29,384 成分 |
| 标注语料（自己标注） | NER/RE 模型训练 | 预计 2000-5000 句 |
| 公开中医 NER/RE 数据集 | baseline 对比 | 视能找到的数据集而定 |

### 6.2 标注方案

使用 Label Studio 标注：
- 实体标注：BIO 标注体系
- 关系标注：实体对 + 关系类型

### 6.3 评价指标

- **NER**：Precision、Recall、F1
- **RE**：Precision、Recall、F1
- **知识图谱规模**：实体数、关系数、三元组数

### 6.4 对比实验

| 方法 | 说明 |
|---|---|
| BERT-base | 基线方法 |
| BERT + 注意力 | 仅加注意力机制 |
| BERT + 药理学特征（我们的方法） | 融合 OB/DL 等特征 |
| BiLSTM-CRF | 经典 NER 方法（NER 对比） |

### 6.5 消融实验

- 去掉药理学特征 → 验证特征融合的有效性
- 去掉注意力机制 → 验证注意力的作用
- 不同特征组合 → 找最优特征子集
