# NER实验结果与改进方案

## 一、首轮实验结果

### 实验配置
- 预训练模型：hfl/chinese-bert-wwm-ext
- 数据：TCM-NER-3000 V2（2332句，7844实体，4类）
- 划分：train=1865, dev=233, test=234
- 训练：10 epochs, batch_size=16, lr=2e-5

### 结果对比

| 模型 | Dev F1 | Test F1 |
|---|---|---|
| BERT-CRF (Baseline) | 0.758 | 0.736 |
| BERT-CRF + SoftLexicon(random) | 0.768 | 0.734 |

### 各实体类型表现

| 类型 | Baseline F1 | SoftLexicon F1 | 测试样本数 |
|---|---|---|---|
| Herb | 0.87 | 0.88 | 417 |
| Formula | 0.83 | 0.82 | 64 |
| Disease | 0.51 | 0.49 | 131 |
| Symptom | 0.52 | 0.54 | 210 |
| Target | 0.00 | 0.00 | 3 |

### 关键发现
1. SoftLexicon(random)几乎无提升，Test F1反而微降0.002
2. **根因分析**：随机初始化的词嵌入是噪声源，拼接到BERT输出上反而污染了特征
3. Herb/F1=0.88（高频+边界明确），Disease/F1=0.51（语义复杂+样本少）
4. Target只有3个测试样本，完全无法学习

## 二、问题定位

SoftLexicon机制的核心：将词典匹配到的词的向量聚合后拼接到BERT字向量上。

**当前困境**：
- BERT = 海量文本预训练的"全能学霸"（字向量质量高）
- 词嵌入 = 随机初始化的"白纸"（无领域知识）
- 拼接 = 低质向量污染高质量特征

**结论**：SoftLexicon路线本身没有失败，失败的是词嵌入质量。

## 三、改进方案：6组递进实验

### 实验设计

| 实验 | 模型 | 目的 | 预期F1 |
|---|---|---|---|
| Exp1 | BERT-CRF | 基线 | 73.6（已有） |
| Exp2 | + SoftLexicon(random) | 消融：词典增强无效 | 73.4（已有） |
| Exp3 | + SoftLexicon(Word2Vec) | 领域词嵌入有效 | 75-77 |
| Exp4 | Domain-adapted BERT-CRF | 领域预训练有效 | 76-78 |
| Exp5 | + SoftLexicon(Word2Vec) | 模型融合 | 77-79 |
| Exp6 | + 数据增强 | 最终模型 | 78-80 |

### 论文叙事逻辑

```
词典知识 → 领域词向量 → 领域语言模型 → 知识增强Pipeline
   ↓              ↓              ↓              ↓
 Exp2失败    Exp3成功       Exp4成功       Exp5+Exp6最终
```

递进关系清晰，审稿人一看就懂。

### 数据处理
- 去掉Target类型（仅3个测试样本），报告4类实体（Herb/Formula/Disease/Symptom）
- 新文件：tcm_ner_3000_4class.bio

## 四、执行计划

### Step 1：删除Target，生成4类BIO数据
- 从tcm_ner_3000.bio中过滤Target标签
- 保持train/dev/test划分一致
- 预计5分钟

### Step 2：训练TCM Word2Vec
- 语料：古籍704本 + TCM_KG节点描述
- 参数：skip-gram, vector_size=200, window=5, min_count=3
- 工具：gensim
- 预计CPU几分钟

### Step 3：替换SoftLexicon词嵌入，跑Exp3
- 加载Word2Vec权重到nn.Embedding.from_pretrained()
- 训练BERT-CRF + SoftLexicon(Word2Vec)
- 预计Kaggle 40分钟

### Step 4：领域BERT继续预训练
- 语料：同Step 2
- 方法：MLM (Masked Language Model)继续预训练
- 工具：transformers Trainer
- 预计Kaggle GPU几小时

### Step 5：跑Exp4/5/6
- Exp4：Domain-adapted BERT-CRF
- Exp5：+ SoftLexicon(Word2Vec)
- Exp6：+ 数据增强（KG模板生成）
