# 数据方案调整：从依赖外部数据到自建数据

## 一、背景

原计划依赖TCMNER2026和DG-SpanTCM两个外部数据集，但：
- TCMNER2026：申请被拒（论文在审稿阶段，暂不公开）
- DG-SpanTCM：邮件未回复

## 二、核心决策

**放弃依赖外部NER数据集，改为自建数据方案。**

### 关键分析

| 任务 | 难度 | 工作量 | 可行性 |
|---|---|---|---|
| 自建NER数据（大规模） | 高 | 几个月 | 不推荐 |
| 自建NER数据（3000句） | 中 | 2-3周 | **可行** |
| 自建RE数据（3000条） | 低 | 1-2周 | **推荐** |

### 为什么NER和RE难度差异大？

**NER**：按字符标注，一句200字符，3000句≈60万字符，工作量大
```
黄 B-CMM
芪 I-CMM
补 B-TCMTM
气 I-TCMTM
...
```

**RE**：按实体对标注，一条几十秒，3000条工作量小很多
```
黄芪
气虚
treats
```

## 三、最终数据方案

### 数据构建流程

```
中医公开文本（古籍、教材、药典）
        │
        ├── TCM_KG词典匹配预标注
        │
        ├── LLM自动标注（GPT/ChatGLM）
        │
        └──────────┬──────────
                   │
            人工审核（30-50%）
                   │
            TCM-NER-3000
                   │
        BERT+SoftLexicon+CRF
                   │
              NER识别结果
                   │
            实体两两配对
                   │
           LLM判断关系+Self-consistency
                   │
            人工审核（20%）
                   │
             TCM-RE-3000
                   │
             RE模型训练
```

### 数据规模

| 数据集 | 规模 | 内容 | 工作量 |
|---|---|---|---|
| TCM-NER-3000 | 3000句 | ≈25000-35000实体，6类 | 词典匹配+LLM标注2-3天，人工审核3-5天 |
| TCM-RE-3000 | 3000条 | 5类关系+no_relation | LLM标注2-3天，人工审核1-2天 |

### 数据来源

| 来源 | 内容 | 用途 |
|---|---|---|
| TCM_KG | 实体词典(~17,000词) + 三元组 | NER词典匹配 + RE种子知识 |
| 中医公开文本 | 古籍、教材、药典 | NER原始语料 |
| TCMNER2026 Demo | 少量中医病历 | 外部验证（不作为主数据） |

## 四、创新点调整

从"模型创新"调整为**"数据+模型协同创新"**：

| 创新点 | 内容 | 贡献 |
|---|---|---|
| 创新一（新增） | 提出基于LLM的中医NER/RE数据自动构建方法 | 数据层面 |
| 创新二（保留） | SoftLexicon词典增强的中医NER | 模型层面 |
| 创新三（保留） | 药理特征+KG先验融合的RE | 模型层面 |

### 论文表述

"针对中医领域标注数据稀缺的问题，提出一种基于LLM的低成本数据构建流程：利用TCM_KG词典进行预标注，结合LLM自动标注与人工审核，构建TCM-NER-3000和TCM-RE-3000数据集。该方法可复现、可扩展，为中医NLP研究提供了数据构建范式。"

## 五、时间规划

| 任务 | 时间 | 产出 |
|---|---|---|
| 收集中医语料 | 2-3天 | 古籍、教材、药典文本 |
| 词典匹配预标注 | 1天 | 预标注结果 |
| LLM自动标注NER | 2-3天 | NER候选标注 |
| 人工审核NER（30-50%） | 3-5天 | TCM-NER-3000 |
| NER模型训练 | 2-3天 | NER模型 |
| LLM自动标注RE | 2-3天 | RE候选标注 |
| 人工审核RE（20%） | 1-2天 | TCM-RE-3000 |
| **总计** | **13-19天** | 完整数据集 |

## 六、风险与应对

| 风险 | 应对 |
|---|---|
| LLM标注质量不稳定 | 人工审核30-50%，不是20% |
| NER错误传递到RE | NER审核比例提高，确保质量 |
| 中医术语LLM理解不准 | 用TCM_KG词典约束，人工校验 |
| 工作量超预期 | 先做小规模验证，再扩展 |
